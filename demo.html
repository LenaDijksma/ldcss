ldcss

A small framework built on two ideas: ld-* classes for structure, data-ld-* attributes for state and variants. No build step. No dependencies.

<button class="ld-btn" data-ld-variant="primary" data-ld-size="lg">
  Run
</button>

ldcss pairs structural classes with state-carrying attributes, so a component's markup reads like a small config file.
