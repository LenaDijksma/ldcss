ldcss

A small framework built on two ideas: ld-* classes for structure, data-ld-* attributes for state and variants. No build step. No dependencies.

<button class="ld-btn" data-ld-variant="primary" data-ld-size="lg">
  Run
</button>

ldcss pairs structural classes with state-carrying attributes, so a component's markup reads like a small config file.

Include ldcss.css and ldcss.js. Everything else is attributes on plain HTML — no build step.

v0.8.0 — added typography (ld-text-{xs..2xl}, ld-font-*, ld-truncate, ld-line-clamp-*), color (ld-text-danger, ld-bg-*), position/z-index (ld-relative/absolute/fixed/sticky, ld-inset-0, ld-z-* incl. named aliases matching this file's own dropdown/modal/toast stacking), flex extras + generic grid (ld-self-*, ld-order-*, ld-grid-cols-*), visibility/interaction (ld-sr-only, ld-cursor-*, ld-select-none, ld-overflow-*), and the first responsive variants (ld-sm:d-*, ld-md:d-*, ld-sm:grid-cols-*, ld-md:grid-cols-*). Lives in ldcss.css §3d–3i.

v0.7.0 — added generated spacing/layout utility classes: ld-pad-{side}-{size} and ld-mar-{side}-{size} (side: a/t/r/b/l/v/h, size: 0–8 mapped to --ld-space-*), plus negative margins (ld-mar-t-n3), ld-gap-*, and display/flex helpers (ld-d-*, ld-flex-*, ld-items-*, ld-justify-*, ld-text-{left,center,right}, ld-w-*, ld-max-w-*). These are real classes in ldcss.css §3b–3c, not a naming convention to remember.

v0.6.0 — pagination is now data-driven: data-ld-paginate="#target" + data-ld-page-size="3" on a .ld-pagination element auto-generates the right number of page buttons from the target's row/item count, and clicking a page actually shows/hides that slice of rows. Manual (hand-written) pagination markup still works as before.

v0.5.2 — bugfix: three components had markup but no wiring. Command palette items now do nothing by default unless given data-ld-command-action — added real actions to the demo. List group items are now clickable and select themselves (data-ld-active) among siblings. Pagination buttons now actually update the active page and disable prev/next at the ends (mark them with data-ld-page="prev"/"next").

v0.5.1 — bugfix: [data-ld-dropzone] defaulted to inline layout when used on a <label>, so its border only wrapped the first line and the filename text spilled outside the box. Now explicitly display: flex.

v0.5.0 — added offcanvas, stepper, command palette (⌘K), range slider, dropzone, rating, tag input. Added ld-scale-* density presets, ld-accent-* per-element color swap, ld-glass/ld-flat surfaces. ARIA wiring (aria-expanded/aria-hidden/aria-selected, dialog roles, focus trap + focus return, live-region toasts) and a prefers-contrast: more pass.

v0.4.0 — added escape-hatch utilities: ld-round*, ld-border-{width}px-{style}, ld-shadow-soft*, ld-font-sans / ld-font-serif — for opting out of the sharp/hard-shadow/monospace defaults per element.

v0.3.0 — added kbd, copy-to-clipboard, terminal window, spinner, toast, table, pagination, list group, switch, avatar, popover, skeleton loader.

v0.2.0 — added alerts, accordion, breadcrumbs, tooltip, progress bar. Added data-ld-notheme to freeze a subtree's theme. Fixed low-contrast code blocks on light theme (new --ld-code-text token).

v0.1.0 — buttons, cards, nav, forms, dropdown, tabs, modal, theming.
